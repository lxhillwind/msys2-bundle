## included packages

see [pkglist.txt](pkglist.txt).

## local setup

- modify mirror (optional)

edit [pacman.conf](pacman.conf) (msys2 mirror) and
[package.sh](package.sh) (alpine mirror).

- run package.sh

will generate `x86_64.7z` containing `usr/` / `etc/`.

## use it

create shortcut to [mintty-start.bat](mintty-start.bat) (which is also
bundled into x86_64.7z: /usr/bin/)
